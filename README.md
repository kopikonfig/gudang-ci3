    - Datatables
    - Chart.js

### Recomended Machine
     - php7.x
     - mysql 8

### Installation
1. define your base_url `/application/config/config`
    ```
    $config['base_url'] = 'http://example.com/path'
    ```
2. define your database address `/application/config/database`
    ```
    'hostname' => 'localhost',
	'username' => 'root',
	'password' => '',
	'database' => 'gudang_app',
    ```
3. restore database
